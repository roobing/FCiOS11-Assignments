- 이름과 나이를 입력 받아 자신을 소개하는 글을 출력하는 함수

  ```swift
  let name: String = ""
  let age: Int = 0
  func introduceMyself(_ name, _ age) {
    print("Hi! My name is \(name) and I am \(age) years old.")
  }
  
  introduceMyself("Woobin", "34")
  ```
